- [ ] Review open PRs across repos
- [ ] Run tests in another repo
- [ ] Update CHANGELOG or docs
- [ ] Check recent commits: `git log --oneline -5`

## Medium Tasks (5-10 min)
- [ ] Write tests for recently completed work
- [ ] Review code Claude just modified
- [ ] Plan next feature in side project
- [ ] Update issue status: `bd show <id>` → `bd comment <id>`

## Context Preservation
- [ ] Set a timer (5-10 min) to check back
- [ ] Keep this terminal visible to see when Claude finishes

## Return Protocol
- [ ] Review what Claude did: read git diff or Claude's summary
- [ ] Test the changes
